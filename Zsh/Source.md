#### Source命令

启动anaconda3的时候，需要通过 source ~/anaconda3/bin/activate root 这条命令进入base环境中

其实它还有另外一个用法，就是: ./a.sh

```shell
# 两种用法,source命令也称为点命令
source filename
. filename  (中间有空格)
```
